# 💻 CV Web - Ignacio Echave

Este proyecto consiste en el desarrollo de un CV web personal utilizando HTML, CSS y JavaScript, con el objetivo de presentar mi perfil profesional, habilidades y proyectos.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git & GitHub

---

## 📌 Funcionalidades

- Diseño moderno y responsive
- Secciones de presentación, skills, servicios y proyectos
- Formulario de contacto funcional
- Botón "Volver al inicio" con scroll suave (JavaScript)
- Uso de variables CSS
- Optimización de imágenes
- Lazy loading para mejorar performance

---

## 🧠 Buenas prácticas aplicadas

Durante el desarrollo se trabajó utilizando un flujo profesional basado en:

- Uso de branches por cada feature/issue
- Creación de Pull Requests
- Refactorización de código
- Uso de unidades relativas (`rem`)
- Uso de clases CSS para mejorar performance
- Nombres descriptivos en variables CSS
